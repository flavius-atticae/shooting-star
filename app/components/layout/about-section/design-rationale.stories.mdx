## ü§± Pregnancy-Safe Design Principles

### **Visual Comfort**
- **Contrast Ratio**: 4.8:1 (#3d5a62 on #f5f4f2) exceeds WCAG AA requirement
- **Color Psychology**: Calming blue reduces anxiety, soft gray prevents eye strain
- **Typography**: 16px+ base size with relaxed line height (1.625) for easy reading
- **Spacing**: Generous whitespace prevents visual claustrophobia

### **Physical Adaptations**
- **Touch Targets**: All interactive elements sized for potentially swollen fingers
- **Reading Comfort**: Optimal line length and spacing for concentration issues
- **No Motion**: Static layout prevents motion-triggered nausea
- **Rounded Elements**: Softer visual appearance reduces stress

### **Cognitive Support**
- **Clear Hierarchy**: Logical information flow reduces mental load
- **Predictable Layout**: Familiar patterns support users with brain fog
- **Simple Language**: Accessible French content for emotional comfort
- **Consistent Spacing**: Rhythm and pattern support visual processing
