## Executive Summary

**Shooting Star** is a professional website for a yoga instructor (Pauline Roussel) specializing in perinatal yoga, doula services, and maternal wellness in Québec. The site targets pregnant women and new mothers in the Montréal area, with all content in French (fr-CA).

The project is a **server-side rendered (SSR) web application** built with modern frontend technologies, emphasizing accessibility (WCAG 2.1 AA), pregnancy-safe UX patterns, and compliance with GDPR, PIPEDA, and Québec Law 25.

## Architecture Pattern

**Component-based Layered Architecture** with server-side rendering:

```
Routes (pages) ─── Server Actions (loaders, actions)
    │                      │
    ▼                      ▼
Layout Components      Server Libs
(Hero, Footer,         (email, honeypot,
 Services, etc.)        rate-limiter)
    │
    ▼
UI Components          Data Layer
(Button, Card,         (static data files
 Section, etc.)         for page content)
    │
    ▼
Design Tokens
(Tailwind theme,
 brand colors)
```

### Key Architecture Decisions

1. **SSR-first** — All pages are server-side rendered for SEO and performance
2. **Component composition** — Pages composed from reusable layout sections
3. **Data co-location** — Page content data stored in `app/data/` TypeScript files
4. **Server-only modules** — Email, honeypot, rate-limiting isolated with `.server.ts` suffix
5. **Progressive enhancement** — Forms work without JavaScript via server actions
6. **Pregnancy-safe UX** — All animations respect `prefers-reduced-motion`, minimum 44px touch targets
