# Pregnancy-Safe UX/UI Guidelines

This documentation details design principles specifically adapted for pregnant women and new mothers, based on perinatal UX research and real user feedback.

### "Pregnancy Brain" - Adapting to Concentration Difficulties

Hormonal changes during pregnancy affect:
- **Short-term memory** : Difficulty retaining multiple pieces of information
- **Selective attention** : Easy distraction
- **Decision making** : Fatigue when making complex choices

**UI Guidelines:**
- âœ… **Simple and predictable navigation**
- âœ… **Progress automatically saved**
- âœ… **Essential information visible without scrolling**
- âœ… **Clear confirmations for important actions**
- âŒ Avoid complex multi-level menus
- âŒ Avoid too many simultaneous options

### Eye Fatigue and Vision

**Common symptoms:**
- Temporary blurred vision
- Dry eyes
- Light sensitivity
- Difficulty with prolonged focus

**Visual Guidelines:**
- âœ… **High contrast (minimum AA, prefer AAA)**
- âœ… **Generous font size (16px+ on mobile)**
- âœ… **Increased line spacing (1.6+)**
- âœ… **Customization options (size, contrast)**

### Pregnancy-Safe Color Palette

#### ğŸš« Colors to Avoid
- **Bright red (#FF0000)** : Association with medical emergencies, anxiety
- **Saturated orange** : Can trigger nausea
- **Fluorescent yellow** : Eye fatigue, migraines
- **Aggressive contrasts** : Excessive stimulation

#### âœ… Recommended Colors
- **Natural greens** : Soothing, associated with nature
- **Soft blues** : Calm, serenity
- **Powder pinks** : Gentleness without aggression
- **Warm neutrals** : Comfort and security

## ğŸ¤± Physical Considerations

### Touch Targets and Dexterity

**Physical changes:**
- **Water retention** : Swollen fingers, less precision
- **Reduced mobility** : Difficulties reaching certain areas
- **Fatigue** : Less energy for complex interactions

**Touch Guidelines:**
- âœ… **Touch targets minimum 44x44px (WCAG)**
- âœ… **Prefer 48x48px or more**
- âœ… **8px minimum spacing between elements**
- âœ… **Extended click area on critical elements**

### Touch-Friendly Example
```tsx
// Pregnancy-optimized button
const PregnancyFriendlyButton = ({ children, ...props }) => (
  <button 
    className="
      min-h-[48px] min-w-[48px]     /* Generous touch target */
      px-6 py-3                     /* Comfortable padding */
      text-lg font-medium           /* Readable text */
      rounded-lg                    /* Soft rounded corners */
      touch-manipulation           /* Touch optimization */
      active:scale-95              /* Subtle visual feedback */
      transition-all duration-150   /* Gentle animation */
    "
    {...props}
  >
    {children}
  </button>
)
```

## ğŸ¤¢ Sensory Considerations

### Nausea and Sensitivity

**Common visual triggers:**
- Fast or jerky animations
- Intense repetitive patterns
- Abrupt color changes
- Aggressive parallax scrolling

**Anti-Nausea Guidelines:**
- âœ… **Slow and fluid animations (duration-300+)**
- âœ… **Respect prefers-reduced-motion**
- âœ… **Smooth transitions between states**
- âœ… **Avoid visually aggressive patterns**

### Gentle Animation Implementation
```tsx
// Pregnancy-safe animation
const gentleTransition = {
  initial: { opacity: 0, y: 20 },
  animate: { opacity: 1, y: 0 },
  transition: { 
    duration: 0.5,      // Slower than standard
    ease: "easeOut"     // Gentle curve
  }
}

// Respect user preferences
const SafeAnimation = ({ children }) => (
  <motion.div 
    {...gentleTransition}
    className="motion-reduce:transition-none motion-reduce:animate-none"
  >
    {children}
  </motion.div>
)
```

## ğŸ“± Device-Specific Optimizations

### Mobile Priority

**Why mobile-first for pregnant women:**
- Usage from bed/couch
- Frequent travel
- Fatigue limiting desktop use
- Emergencies requiring quick access

### Tablet Optimization

**Tablets particularly appreciated:**
- Comfortable screen without neck strain
- Possible lying down usage
- More comfortable long reading
- Sufficient tactile precision despite water retention
