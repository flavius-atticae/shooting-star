Welcome to the Shooting Star design system for **Pauline Roussel Yoga & Perinatal Accompaniment**.

This foundational component collection is specifically designed to create **pregnancy-safe** digital experiences adapted for pregnant women and new mothers in Quebec.

## ğŸ§± Foundation Components

### Container
Maximum width and responsive padding system optimized for different content types.

**Available Sizes:**
- `sm` : Focused content (articles, forms)
- `md` : General usage (default)
- `lg` : Complex layouts
- `xl` : Extended interfaces
- `full` : Full width with padding

### Background
Pregnancy-safe backgrounds using the Pauline Roussel color palette.

**Available Variants:**
- `white` : Neutral base
- `accent` : Soothing light gray  
- `soft` : Gentle gradient
- `gradient-soft` : Gradient with pale rose
- `gradient-warm` : Warm gradient

### AdaptiveGrid
Responsive grid system using Container Queries with intelligent fallback.

**Configurations:**
- **2 columns** : Ideal for services, testimonials
- **3 columns** : Galleries, categories on large screens
- **Configurable gap** : Adaptive spacing
- **Automatic fallback** : Media queries if Container Queries not supported
